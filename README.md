# DL Farm Management System

A complete farm management system built with Python Flask.

## Features
- ✅ 3 User Roles: Admin, Manager, Accountant
- ✅ Complete Animal Management
- ✅ Breeding & Medical Records
- ✅ Sales & Financial Management
- ✅ Inventory & Feed Management
- ✅ Staff & Task Management
- ✅ Responsive Web Interface
- ✅ Charts & Analytics
- ✅ Role-Based Access Control

## Installation

### Method 1: One-Click Setup (Recommended)
1. Make sure Python 3.8+ is installed
2. Run: `python deploy.py`
3. System will install dependencies and start automatically

### Method 2: Manual Setup
```bash
# 1. Clone/download the project
# 2. Install requirements
pip install -r requirements.txt

# 3. Seed database (optional)
python seed_data.py

# 4. Run the application
python app.py
